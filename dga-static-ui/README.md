# DGA 分析智能体后台 · 静态 UI 原型

面向毕业设计《变压器油中溶解气体分析智能体研究与实现》的高保真静态 HTML 原型，依据根目录 `重构设计-DGA分析智能体后台.md` 设计。

## 打开方式

直接用浏览器打开 `index.html`，或本地起一个静态服务：

```bash
cd dga-static-ui
python3 -m http.server 8080
# 访问 http://localhost:8080
```

## 页面分工（与重构设计对齐）

| 页面 | 文档对应 | 运维看什么 | 不做什么 |
|------|----------|------------|----------|
| `index.html` | 展示总览 | 360 天整体态势、图表 | 不重讲各模块结论 |
| `detection.html` | 模块① **检测=分级** | 判定结论 + 表 A.3 三项 + §9.3.3 | 仅分档三气体，五烃在类型判断页 |
| `trend.html` | 模块③ **趋势辅线** | 近90天速率 +「预」事件表 + 速率计算 | 不重判四档 |
| `alerts.html` | 模块② **告警面板** | 分页表格 + 查看 G.1 报告 | 不重算档位（分级只算一次） |
| `diagnosis.html` | 模块 **故障类型判断** | 三种国标方法 ①②③ + 一致性比对 | 注意值 2+ 才触发 |
| `agent.html` | 模块④ Agent | 编排 + G.1 报告 | — |

**产气速率两处分工（文档原文）**：气体分级页用速率判「急不急」；产气趋势页展示完整曲线 +「预」。
