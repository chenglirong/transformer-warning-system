## 系统一句话

面向 220kV 及以下油浸式变压器,基于 DGA 的**分析智能体后台软件**(对标国标站端后台):检测(分级)+ 故障类型判断 + 分级告警 + 趋势预警;创新点 = **智能体(Agent)编排** 整个分析—预警流程。

- **题目**:《变压器油中溶解气体分析智能体研究与实现》
- **数据**:全程跑合成 360 天时序(原始 743 快照仅作合成分布原料);**7 种特征气体全用**(H₂/CH₄/C₂H₄/C₂H₆/C₂H₂/CO/CO₂)
- **输出定位**:依 DL/T 722 输出检测分级 + **故障类型判断结论 + 措施建议**(附录D / 1685 附录B 试验映射);**不做健康评分**(1685 扣分 / 984)、**不给运维处置指令**(只到试验层),成因/部位定位归展望

## 实施顺序(不锁时间,先夯地基)

1. **数据地基**:`synthesis.py` 对齐对数正态/相关性 + 7 气全生成
2. **检测+告警**:表A.3 四档全报 + §9.3.3 处置解读(不改档位)+ 两套速率(D-004)
3. **故障类型判断**:三方法融合;触发=注意值2 **或** 722 产气速率超注意(与研判双门槛)
4. **趋势**:722 §9.3.2 %/月 +「预」
5. **Agent**:编排 + B 报告 + C 决策(在线周期 A.3.1≤12h / §5.5.5≤2h)
6. **前端 + 联调**
7. **收尾**:论文从零重梳

## 关键约束(实现时守住)

- **算法层与 Web/DB 解耦**:`algorithms/` 输入输出都是 DataFrame,不碰 HTTP/DB
- **数值以国标原图为准**
- **检测**:表A.3 分级是事实档位;§9.3.3 只调紧急度不改档位
- **判型触发**:注意值2+ **或** 722 相对产气速率连续超注意 → 进判型;处置研判仍仅注意值2+
- **Agent C**:在线采集周期(A.3.1 / §5.4.5 / §5.5.5);722 表1/§5.4b 仅离线对照
- **不做**:时序预测模型、健康评分、运维处置指令、成因/部位定位
