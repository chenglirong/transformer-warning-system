## 系统一句话

面向 220kV 及以下油浸式变压器,基于 DGA 的**分析智能体后台软件**:检测(分级) + 故障类型判断 + 分级告警 + 产气趋势/涨势预警;创新点 = **智能体(Agent)编排** 整个分析流程。

- **题目**:《变压器油中溶解气体分析智能体研究与实现》
- **数据**:全程跑合成 360 天时序(原始 743 快照仅作合成分布原料);**7 气全生成**(表A.3 落档只用乙炔/H₂/总烃;CO/CO₂ 走表5/CO₂/CO)
- **输出定位**:依 DL/T 722 / 1498.2 / 1685 输出档位、故障类型、措施建议(附录D / 1685 附录B 试验映射)、附录 G 档案卡片;**不做健康评分**、**不给运维处置指令**(只到试验层),成因/部位定位归展望

## Agent 七步顺序

当日气体 → 四档分级 → **产气趋势** → 处置紧急度 → 故障类型 → 监测决策 → 生成报告

- **判型触发**:注意值2/告警 **或** 总烃月环比超注意(含涨势预警);处置紧急度仍仅注意值2+
- **用词**:步骤名「产气趋势」;结果态「涨势预警」(档未达注意值2) / 「涨势快」 / 「未超注意值」
- **LLM**:只写分析意见与 G.2 试验成稿人话,不改判定;无密钥或失败 → 规则模板

## 关键约束

- **算法层与 Web/DB 解耦**:`algorithms/` 不碰 HTTP/DB
- **数值以国标原图为准**
- **两套速率各归各**(D-004):表A.3 %/周只落档;722 §9.3.2 %/月管紧急度 / 趋势 / 涨势预警
- **绝对产气率(mL/天)**:式1 需油重,合成无铭牌 → 报告该格填「—」
- **不做**:时序预测、健康评分、运维处置指令、成因/部位定位、自由 ReAct

