模块 0 · 数据预处理

必做 3 步 洗数据

  • 类型清洗：CO/CO₂ 列 '-'（未测）→ NaN，列恢复数值类型
  • 选列：核心取 5 种烃类，CO/CO₂ 作辅助列保留；弃 H₂O（全空）/O₂/N₂
  • 缺失标注：CO 缺≈22% / CO₂ 缺≈31%；不强行填满，有值则用、无值则跳过（诚实原则）

造特征 全程在合成360天上

  • 气体比值（三比值）+ 总烃
  • Duval 三角坐标（供分级 / 诊断）
  • 产气速率（一阶差分，供检测涨势佐证 + 模块3趋势）
  • 删滑窗统计（不做 LSTM）、删油温速率（无油温字段）
沿用现有 features.py 改造

数据策略：真实快照 + 合成时序

原始 743 快照 bmrayan 数据集

  • 5 种烃类 100% 完整；CO 78% / CO₂ 69% 有值（辅助）
  • 真实 μL/L 量级，18 条故障标签
  • 角色：合成的分布原料 + 答辩诚实交代来源

  • 单台最干净好讲（原始无设备标识，多台更假）
  • 提供时间维度 → 可算产气速率 =「预」的来源

  • ▸ 超注意值但长期稳定 → 档位仍报，处置=「加强监视，不紧急」
  • ▸ 未超但产气速率超 → 档位报正常，处置=「涨势快，缩短周期」（即「预」）
  • ▸ 仅 H₂ 超但不涨 → 档位如实报，处置=「大概率非故障，加强监视」（不推翻档位）

模块 1-① 处置紧急度研判产气速率视角，供 Agent 判「急不急、下一步动作」（仅注意值2以上启动）

这一步不判「是不是异常」，也不改分级档位（档位由表A.3 定，是事实、不被推翻）。它回答的是「这个异常急不急、要不要采取动作」—— 依据 §9.3.3 a)「气体含量超过注意值时，应结合产气速率进行判断；超过注意值但长期稳定，可继续运行」。结论喂给 Agent 做处置决策（缩周期 / 二次采样 / 继续监视），不动检测档位。

产气速率判据 · 涨势视角

  • 看：涨得快不快（超没超注意值）
  • 依据：DL/T 722 §9.3.3 a)「结合产气速率判断」（需时序，跑在合成数据上）
  • 产气速率两处分工：此处判涨势超没超→供处置紧急度研判；模块3做完整趋势展示 +「预」
为什么从注意值2 启动　注意值2（C₂H₂≥5 / H₂≥150 / 总烃≥150）正是 DL/T 722 表3 的经典注意值线（表A.3 注意值2 = 722 表3，同一条线、两标准叫法）；§9.3.3 a) 明文「超过注意值才结合产气速率判断」，故门槛卡在注意值2，有国标原文撑腰。产气速率与表A.3 互不重叠（表A.3看绝对值、产气速率看变化率），故不是「再确认一遍超没超」而是国标要的「据产气速率定处置」。

辅助诊断的触发时机与局限

触发时机 §10.3 + §10.2.4 a)

  • §10.3 判断步骤：先比表3注意值 → 认为有故障 → 才用表5/6/7 判故障类型
  • 而表3注意值 = 分级的注意值2（同一条线两标准叫法）
  • → 辅助诊断仅对注意值2及以上触发，与处置紧急度研判同一条线；注意值1 及正常档只报档位不给倾向
  • §10.2.4 a) 印证：比值法只在超注意值后才有效，正常数据「比值法没有意义」

低浓度局限 §10.2.4 c)

  • 气体浓度 <10μL/L 时，比值法误差大（比值误差已达 20%+）
  • → 低浓度样本的辅助倾向标可信度低

为什么不做故障确诊 双重依据

  • 数据：bmrayan 真故障仅 4 条，无法验证多分类准确率
  • 国标：附录D 表D.1 明文——DGA 判出故障后，须结合绕组电阻/绝缘油/局放测量等停电电气试验才能定位成因
  • → DGA 到「过热/放电 + 六分区」这层到顶，成因/确诊 留作展望
三方法怎么比一致性　三比值 + 大卫三角本是同一套六代码（PD/D1/D2/T1/T2/T3）直接比落格；特征气体法（表5，按油/纸分）保留原话不硬翻代码（国标无官方映射，硬映射=无依据）。三方一致性统一拉到「放电 vs 过热」性质大类比——都指同一大类→倾向可信度高，打架→标低。粒度不同不是缺陷，恰体现「多方法各有侧重」。

模块 4 · 智能体

为什么非 Agent 不可：模块1-3（分级/研判/诊断/趋势）全是确定性方法（查表/投票/算数），规则代码就能做。Agent 的不可替代价值只在「需要判断或表达」处 —— 分析层保证「准」，智能体层保证「智能」。

B、生成自然语言分析报告 必做，LLM 表达价值

  • 把多维结果综合成一段人话报告
  • 内容：预警级别 + 超标项 + 趋势 + 处置紧急度研判（产气速率）+ 辅助倾向（倾向XXX）+ 建议动作
  • 版式套 DL/T 722 附录G 表G.1「油中溶解气体分析档案卡片」，报告落「分析意见」栏
  • 合成拿不到的字段（铭牌/工况/O₂·N₂/绝对产气率）诚实留空「—」不杜撰（P1）
  • 叫「分析报告」

C、自主决策下一步动作 体现「有决策」

  • 研判紧急度低/诊断三方分歧 → 建议二次采样验证（§5.4.5）
  • 异常时缩短周期（§5.4 b）：过热→周 / 放电→天，按倾向触发非确诊
  • 只出监测决策（采样/调周期），不出运维处置（停电/检修）
自由对话问答 — 考虑（幻觉/易越界/不可控）；用「固定按钮 + B式受控生成」替代。
