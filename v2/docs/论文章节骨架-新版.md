# 论文章节骨架——新版

> 题目：**变压器油中溶解气体分析智能体研究与实现**
>
> **写作边界**（与 `v2` 实现一致）：面向 220kV 及以下油浸式变压器在线 DGA 后台；主线为 1498.2 表A.3 四档、722 产气速率与涨势预警、故障判型、试验建议、附录 G 报告；Agent 编排全流程，LLM 只写报告、不改档位/判型/阈值。不做健康评分、检修/停运指令、成因定位、时序预测。判据不单列专章，靠绪论总览图 + 各章引用 + 附录一对照表。
>
> **写作风格**：算法章（2–4）「引言 → 相关研究与问题分析 → 方法 → 方法验证与案例分析 → 小结」；系统章（5）直接写架构/模块/库表/界面/演示；合成时序口径为**大模型生成**，正文不展开生成细节（细节见附录三）。
>
> **数据口径（全文统一）**：验证与演示**共用一套**「单台虚拟设备 × 360 天」合成时序；原始 743 快照仅作合成分布原料，**不**再单独开一套「快照验证 / 时序验证」双轨。第 2/3/4/5 章凡写「测试数据集」均指该套时序（或从中取样日）。
>
> **相关研究写法**（各章 2.2/3.2/4.2）：按问题维度分类，每类 2–3 篇代表作，末尾点出现有方法未解决问题以引出本章；避免逐篇堆砌「XX年XX提出…」。
